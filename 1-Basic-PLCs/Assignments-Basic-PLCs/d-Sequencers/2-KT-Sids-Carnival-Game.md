# KT & SID'S CARNIVAL GAME v1.0

## WORK ORDER:
Create a carnival game (reaction/timing) using the following instructions:

- (1) TON TIMER instructions 
- (1) SQO SEQUENCER OUTPUT instruction 
- All necessary MOV instructions 
- All necessary XIC, XIO and OTE instructions 

Other instructions may be used with instructor approval.

## OBJECTIVE:
- Press a button to stop the light in the middle (winner!) 


## PARAMETERS:
- Properly configured I/O Modules (emulated and lab) are required. 
- Properly wire (2) push buttons to the input module (Stop and Start). 
- Properly wire (**5** or 7) lights to the output module:
    - The middle light cover will be a different color. 
    - All other light covers will be the same color. 
- Ensure the lights have the proper rating (24VDC LED bulbs).
- *Proper documentation and rung comments are mandatory. 
- *Lab safety is non-negotiable.
