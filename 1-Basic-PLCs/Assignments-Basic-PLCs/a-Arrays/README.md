# ARRAYS

Arrays are a way to store data of the same type, in order to save crucial memory space

>Think of data as old fashion snail mail

An array (PLC memory) is a city street with several hotels located along its length

We need to know where to send (store) this mail:
- The first dimension tells us which hotel on that street
- The second dimension tells us which floor in that hotel
- The third dimension tells us which door on that floor in that hotel
