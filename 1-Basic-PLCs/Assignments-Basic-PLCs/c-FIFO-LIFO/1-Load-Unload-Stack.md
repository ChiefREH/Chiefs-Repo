# LOAD/UNLOAD - "STACK OF ONES"

## CODE:
Create a routine that will move the number 1 **to and from** an Array using one of the following methods:
- FIFO - "First In, First Out" instruction 
- LIFO - "Last In, First Out" instruction 
 
## PARAMETERS:
- All necessary XIC and XIO instructions 
- TON instructions as needed
- CTU instructions as needed
- FFL instructions as needed
- FFU instructions as needed
- ONS instructions as needed
- Properly created Array(s)
- Math instructions as needed
- Comparison instructions as needed
- MOV instructions as needed 
- COP instructions as needed
- CLR instructions as needed 
- RES instructions as needed 
- BRANCHES as needed 
 
## NOTES:
- The Source should be a tag that stores the number 1
- Operate the routine with a seal-in circuit
 
*Proper rung documentation is mandatory.
