# Cannon Ball 1 (MSG)

_Students work in teams of 2_

- Create a routine to activate a remote PLC using Ethernet communications
- Properly configured MSG instruction 
- Properly configured Controller scope tags versus Local scope tags 
- All necessary graphics and HMI configurations 

## PARAMETERS:

- MSG PATHWAY (LAB): 2, target IP address, 1, 0 
- MSG TYPE: CIP Data Table Write 
    - _Does MSG like sending BOOLS?_
    - _is there a work-around using DINT + EQU instructions?_
