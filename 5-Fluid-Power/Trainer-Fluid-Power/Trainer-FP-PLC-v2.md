# TRAINER FP PROGRAM v2.0

Working in your assigned teams, design and deploy a pneumatic control system using:

- (1) Programmable Logic Controller (any brand/language) 
- (2) 24VDC Control Valves 
- (2) pneumatic Cylinders 
- (1) air Regulator 
- (2) push buttons to Stop/Start the program 
- all necessary tubing and fittings 

If we have parts and time:
- (1) proximity sensor for the Cylinder 
- (1) pressure switch for the system 
- (2) lights or LEDs for visual feedback 

## NOTE:

- You may be tasked with additional components/programming.
- Use the minimum supply pressure to achieve the desired operation.

## PARAMETERS:

The following criteria must be met:
- Control routine must operate in a continuous loop ("extend - pause - retract - pause") 

For a Two Cylinder Sequence: 
1. cylinder 1 extend 
2. pause 2 sec 
3. cylinder 2 extend 
4. pause 2 sec 
5. cylinder 1 retract 
6. pause 2 sec 
7. cylinder 2 retract 
8. pause 2 sec 
9. goto step 1 

## INSTALLATION:

Perform installation of all necessary hardware:
- Fabricate and/or install mounting hardware 
- Route wires for all components 
- Make electrical connections in accordance with acceptable standards 

## GRADING:

- Final installation must be neat, clean and professional 
- All documentation, simple schematics, and final code are required for Instructor review 
- Schematics must be legible. No crayon drawings will be accepted.
