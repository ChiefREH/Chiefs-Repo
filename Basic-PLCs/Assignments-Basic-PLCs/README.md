# WORK PARAMETERS

### WARNING: NEVER INSERT OR REMOVE FUSES WITH YOUR FINGERS!

## AT THE START OF LAB
- Router is plugged in
- PLC is plugged in
- HMI has booted up
- 208V AC power disconnect is CLOSED (for VFD work)
- Fuses are installed in the 208V AC main disconnect
- Ping your components for networking (if needed)

## AT THE END OF LAB
- PLC and HMI are unplugged
- 208V AC disconnect is OPEN
- Fuses are removed from the 208V AC main disconnect
- The network router is unplugged
- All computers are logged off/signed out
- No other equipment remains powered up or running

## PROGRAMMING NOTES

- Proper rung documentation is mandatory.
- All documentation must be available on request.
- These NOT a group exercises. Do your own work! 
- Safety in the lab is mandatory! 



_When in doubt, ask the Chief._
