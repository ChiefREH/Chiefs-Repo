# PARALLEL WIRING

Complete the following tasks:
- Properly wire (2) circuits in PARALLEL from the TRANSFORMER secondary (X1-X2): 
    - (1) N.C. momentary push-button 
    - (1) N.O. momentary push-button 
- Properly wire a single LIGHT to each push-button 
- Properly install the FUSES into the MAIN POWER DISCONNECT box 

*Troubleshoot as needed

NOTE: X1-F1 is made via the FUSE
