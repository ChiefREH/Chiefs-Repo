# MANUAL ROBOT CONTROL

## TASK
- Safely operate the Robot using the Teach Pendant.
- Utilize the “handles” function in Roboguide.
 
## FOCUS
- Adjust the SPEED on the Teach Pendent
- Stroke Limit/Limit Error for each axis.
- Avoid putting the Robot into "SINGULARITY"

## JOGGING IN JOINT MODE
- Move all Joints manually (J1 – J6)
- Return to 0.0 using POSN (bottom) / JOINT view
- Reset to 0.0 using the POSN button (right side TP, Roboguide only)
 
## MOVING IN WORLD MODE
- X, Y, Z, Roll, Pitch, Yaw
- "Right-Hand Rule"
