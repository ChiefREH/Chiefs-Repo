# NEW SIMULATION PROJECT

You will use these settings every time you create a new Roboguide work-cell project

## PARAMETERS

Accept *most defaults with these exceptions:
- Process: HandlingPRO
- Creation Method: New robot with default HandlingPRO config.
- Application/Tool: Set EOAT later
- Robot Model: H755 - LR Mate 200iD
- Additional Motion Groups: Accept the defaults by clicking Next
- Robot Options: Accept the defaults by clicking Next
- Summary: Click Finish to create your world

_*Some choices will vary by software version._
