# AOI TEMPERATURE CONVERSIONS

Use Add-On Instructions (AOI) to create the following conversions:

- Convert Fahrenheit to Celsius
- Convert Celsius to Fahrenheit

## HMI PARAMETERS:

- Include numeric input and numeric displays
- Include instructions to reset/restart routine
