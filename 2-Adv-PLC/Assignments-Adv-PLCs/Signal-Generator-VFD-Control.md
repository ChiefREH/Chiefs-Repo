# Local/Remote VFD Speed Control w/ Signal Generator

>_What happens when the VFD potentiometer goes bad? We improvise, we adapt!_

Create a simple **local** and **remote** VFD control panel using the following components:
- Analog I/O modules
- Signal Generator
- PowerFlex 525 VFD
 
## LOCAL CONTROL PARAMETERS

- Control the local VFD Coil (power) from the HMI
- Control the local VFD Run mode from the HMI
- Control the local VFD **Frequency** with the Signal Generator
- Control only the remote VFD, not remote and local at the same time!
- Display all relevant information on the HMI screen
    - Volts, Amps, Freq, GSV data, etc.

## REMOTE CONTROL PARAMETERS

- Control the remote VFD "Run mode" from the HMI
    - Run mode only, **not** the remote Coil!
- Control the remote VFD **Frequency** with the Signal Generator
- Can use MSG instructions, Produced/Consumed tags or other method to achieve results
