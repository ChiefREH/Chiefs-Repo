# Automatic Sequencer

- Use any and all components to create a hydraulic circuit with 3 Pistons.

## Automate the circuit's operation to perform the following sequence IN THIS ORDER:

- Fully extend Piston #1
  - Fully extend Piston #2
    - Fully extend Piston #3
- Fully retract Piston #1
  - Fully retract Piston #2
    - Fully retract Piston #3
- Repeat

### Note: Once started, the automation must loop continuously.
