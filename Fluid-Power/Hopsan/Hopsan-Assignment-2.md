# Create an automated simulation using the following:

- 4/2 Directional Control Valve
- C-type Source
- C-type Tank
- C-type Piston
- Translational Mass
- Force Source

## ER signal loop

- Extend the Piston for 5 seconds
- Retract the Piston for 5 seconds
  - Repeat continuously
